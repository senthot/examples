Senthot sample library

[Installasi]
Untuk menggunakan contoh, harus terlebih dahulu menginstal Senthot (https://github.com/senthot/framework), kemudian masukkan direktori senthot kedalam direktori exsamples, impor file database SQL dan modifikasi informasi file config.php
